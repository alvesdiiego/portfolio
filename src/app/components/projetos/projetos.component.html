Projetos

Cálculos JS

O projeto basicamente é uma calculadora, que realiza operações básicas de mátematica e a tabuada. Projeto desenvolvido com JAVASCRIPT, CSS e HTML.

Visitar link:https://alvesdiiego.github.io/calculadoraJS/

Bankline APP

O Projeto simula um banco real, onde é possivel fazer cadastros, consultar extratos, realizar operações bancarias. O progama é divido em duas partes, front-end que foi desenvolvido em ANGULAR, o back-end em JAVA com SPRING FRAMEWORK e POSTGRESQL.

Visitar link:https://appbankline.netlify.app/

Pesquisa Github

O projeto é uma plataforma de pesquisa de usuarios do github, onde é possivel consultar algumas informações do usuario. O front-end foi desenvolvido em REACTJS, e consumindo a própria API do Github.

Visitar link:https://pesquisagh.netlify.app/
